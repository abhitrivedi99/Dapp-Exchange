# Blockchain Exchange DAPP

This is Practical tutorial of Exchange token Dapp.

## Dependencies

```
- Solidity Compiler
- Truffle
- Ganache
- NodeJS
```

### Deployment of smart contarct

```
- Open Ganache
- truffle migrate --reset
- truffle exec scripts/seed-exchange.js
```

### Test smart contarct

```
- Open Ganache
- truffle test
```
