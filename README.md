# DAPP Exchange

This is Practical tutorial of Exchange token Dapp with React.

## Dependencies

```
- Solidity Compiler
- Truffle
- Ganache
- NodeJS
- Metamask
```

### Deployment of smart contarct

```
- Open Ganache
- truffle migrate --reset
- truffle exec scripts/seed-exchange.js
```

### Test smart contarct

```
- Open Ganache
- truffle test
```
